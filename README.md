# ScholarStream - Student Portal

## About
ScholarStream is a comprehensive School Management System (SMS) element designed to streamline the student experience. It serves as a central hub for students to manage their academic life, providing easy access to essential information and tools.

## Key Benefits
*   **Centralized Information:** Access all enrolled classes, current grades, and assignments in one unified dashboard.
*   **Improved Organization:** Specific trackers for upcoming assignments help students stay on top of deadlines and manage their time effectively.
*   **Transparency:** Real-time visibility into academic performance, including current GPA and detailed grade breakdowns.
*   **Accessibility:** A web-based interface ensures students can check announcements and course materials from anywhere.

## Tech Stack
*   **HTML5**
*   **CSS3**
*   **JavaScript**

## How to Run
1.  Clone the repository or download the files.
2.  Open `index.html` in your web browser.
